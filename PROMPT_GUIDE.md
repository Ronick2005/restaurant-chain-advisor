# Prompt Guide

Effective prompts for different user roles to get the best results from the Restaurant Chain Advisor.

## General Tips

- Be specific about location (city and area)
- Mention cuisine type for better recommendations
- Ask follow-up questions for more details
- Request sources if you need document references

## Tips for Better Results

1. **Be Specific**: "Italian restaurant in T Nagar, Chennai" is better than "restaurant in Chennai"

2. **Provide Context**: Mention target demographic, budget, concept (casual/fine dining)

3. **Ask Follow-ups**: System maintains conversation context for better responses

4. **Request Sources**: If you need document references, they're automatically included

5. **Combine Topics**: Ask about location, market, and regulations together for comprehensive analysis

6. **Use Comparisons**: Compare multiple areas to make informed decisions

7. **Specify Requirements**: Mention alcohol service, seating capacity, parking needs, etc.

---

## Avoid These

- ❌ Vague queries: "Tell me about restaurants"
- ❌ Too broad: "All information about Mumbai"
- ❌ Multiple unrelated topics in one query
- ❌ Queries about cities outside India
- ❌ Personal opinions ("Do you think...")

## Preferred

- ✓ Specific queries with city/area
- ✓ Clear intent (location/market/regulatory)
- ✓ Follow-up questions for details
- ✓ Queries focused on Indian cities
- ✓ Data-driven questions
